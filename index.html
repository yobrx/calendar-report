Configuration

Entrez votre Google OAuth 2.0 Client ID.
Le calendrier et les libellés seront définis à l'étape suivante.

Comment obtenir un Client ID ?
  1. Ouvrez Google Cloud Console
  2. Créez un projet puis activez l'API Google Calendar
  3. Identifiants → Créer → ID client OAuth 2.0 → Application Web
  4. Origines JS autorisées : ajoutez https://votre-user.github.io
  5. Copiez le Client ID et collez-le ci-dessus

Calendrier

Sélectionnez dans la liste ou saisissez manuellement. Utilisez primary pour votre calendrier principal.

La semaine en cours (incomplète) est toujours exclue.

Absences (outOfOffice)

Chaque jour ouvré (lun–ven) marqué comme absence complète sera compté comme 7h dans le libellé ci-dessous. Laissez vide pour ignorer.

Couleurs → Libellés

Couleurs récupérées depuis l'API Google Calendar, triées par nombre d'utilisations sur les 2 derniers mois. Saisissez librement le libellé de chaque couleur. Cochez Exclure pour ne pas comptabiliser ce libellé dans le rapport. Laissez le libellé vide pour ignorer entièrement cette couleur.
